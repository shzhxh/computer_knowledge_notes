#### 属性

语法：

```rust
#![Attr]	// 内部属性，作用于声明属性的地方
#[Attr]		// 外部属性，作用于属性之后的东西
```

分类：

- 内建属性
- 宏属性
- 派生宏辅助属性
- 工具属性
