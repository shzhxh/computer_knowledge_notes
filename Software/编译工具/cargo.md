#### 简介

cargo是rust的包管理器。

#### 用法

`cargo [options] <command> [args]`

#### 命令

```bash
# 构建命令

# manifest命令

# 包命令
new	# 创建一个cargo包
	--bin	# 默认选项。包里会有src/main.rs(二进制目标)。
	--lib	# 包里会有src/lib.rs(库目标)。
# publishing命令

# 通用命令
help
version
```
