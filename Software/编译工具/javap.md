#### 简介

反汇编class文件

#### 语法

```
javap [选项] [classfile]
```

#### 选项

```
-help, --help, -?	# 打印帮助信息
-version
-l					# 打印行和本地变量表
-public				# 只显示公共类和成员
-protected			# 只显示protected和public类和成员
-p, -private		# 显示所有变量和成员
-Joption			# 将option传递给JVM
-s	
-ssysinfo
-constants
-c					# 打印反汇编代码
-verbose
-classpath path
-bootclasspath path
-extdir dirs
```
