#### 简介

编码或解码。注，Ubuntu18.04下无此命令。

#### 语法

```
basenc [options] [file]
```

#### 选项

```
--base64			# 相当于base64命令
--base64url			# file-safe和url-safe base64
--base32			# 相当于base32命令
--base32hex			# 扩展的hex字母表base32
--base16
--base2msbf
--base2lsbf
-d, --decode		# 解码数据
-i, --ignore-garbage
-w, --wrap=COLS
--z85
--help
--version
```
