打印或修改终端的属性

```
stty <-F DEV> [SETINGS]
stty <-F DEV> [-a | -g]	# 打印当前设置
```

选项

```
-a, --all			# 以人类易读的方式打印设置
-g, --save			# 以stty易读的方式打印设置
-F, --file=<DEV>	# 指定设备。默认stdin
--help
--version
```
