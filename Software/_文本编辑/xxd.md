```
xxd [options] [infile [outfile]]
```

选项

```
-a, -autoskip			# 用'*'代替空行。
-b
-c, -cols <cols>		# 强制每行<cols>个字节。默认16，最大可以是256。
-C
-E
-e
-g <bytes>
-h
-i
-l, -len <len>		# 在写了<len>个字节后停止。
-o <offset>
-p, -ps			# 仅输出16进制，即plain hexdump形式。
-r, -revert		# 把16进制转化为2进制。
-seek <offset>
-s [+|-] <seek>	# 从文件内的<seek>偏移处开始。
-u
-v
```
