##### 基本启动

##### 日志和输入文件

```
-q, --quiet		# 关闭wget的输出
```



##### 下载

```
-c, --continue	# 继续获取下载到一半的文件。
-T <sec>, --timeout=<sec>	# 设置网络的timeout为<sec>秒。
-t <num>, --tries=<num>		# 尝试的次数设置为<num>。
-O <file>, --output-document=<file>	# 文档不会写入到相应的文件，而是把所有的文档连接起来写入到<file>。
```
