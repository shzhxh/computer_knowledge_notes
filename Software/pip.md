#### 简介

python包的管理工具

#### 语法

```
pip <command> [options]		# python2的包管理工具
pip3 <command> [options]	# python3的包管理工具
```

#### 命令

```
help
install
uninstall
freeze
show
search
wheel
```

#### 选项

##### 通用选项

##### install选项

```
-e, --editable <path/url>	# 以可编辑的模式，从本地path或远程url来安装一个项目。
-U, --upgrade	# 把所有的包都更新到最新的版本。
--pre			# 默认情况下，pip只寻找稳定版。使用此选项后，将会包含预发布版和开发版。
```
